ShiftLog
  ShiftLog is a sleek, high-performance web application designed for workers to track their shifts, calculate earnings in real-time, and manage work-related expenses like transportation. Built with a focus on speed and a modern aesthetic, it provides a comprehensive dashboard for financial and schedule management.

✨ Features
  Real-time Earnings Calculation: Instantly see your projected pay based on hourly rates, night shifts, and transport allowances.
  
  Multi-Date Entry: Quickly add the same shift details across multiple dates using the built-in mini-calendar.
  
  Comprehensive Analytics:

   Monthly View: A full calendar display showing shifts, totals, and daily breakdowns.
    
   Interactive Graphs: Visualize your income and hours worked over time.
    
   Statistics Dashboard: Track total earnings, hours, and net pay at a glance.

  Dark & Light Modes: Includes a high-contrast dark theme and a clean light theme for comfortable use in any environment.
  
  Mobile First Design: Fully responsive interface with a dedicated bottom navigation bar for mobile users.
  
  Offline Support: Visual indicators for connectivity status and local-first data handling.
  
  Multilingual Support: Toggle between multiple languages (e.g., English, Japanese).

🛠️ Technology Stack
  Frontend: Pure HTML5, CSS3 (using Syne and JetBrains Mono fonts), and Vanilla JavaScript.
  
  Data Visualization: Chart.js for interactive reporting.
  
  Storage: IndexedDB (local browser storage) for persistence without a backend.
  
  Styling: Modern CSS variables, flexbox/grid layouts, and smooth keyframe animations.

🚀 Getting Started
  Since ShiftLog is a client-side application, you can run it directly in any modern web browser.
  
  Clone the repository:
  
    Bash
    git clone https://github.com/yourusername/shiftlog.git
  Open index.html in your favorite browser.
  
  (Optional) Host it on GitHub Pages or any static site provider for easy access.

📱 Usage
  Add Works: Go to the 'Works' tab to define your different jobs and their respective hourly rates.
  ![Visitor Count](https://komarev.com/ghpvc/?username=KaungSetLin3013)
  
  Log Shifts: Use the 'Add' tab to enter shift times. You can select multiple dates at once for recurring schedules.
  
  View Stats: Check the 'Monthly' and 'Stats' tabs to see your financial progress and work history.
