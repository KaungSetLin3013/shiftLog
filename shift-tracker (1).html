Data & Backup

All data is saved in your browser's IndexedDB — it stays on this device permanently and works completely offline. Use the tools below to back up or restore your data.

Export Backup
Download all your works and shifts as a JSON file. Save this file somewhere safe as a backup.
Import Backup
Restore data from a previously exported JSON backup file. This will replace all current data.
Export CSV
Export all shifts as a CSV spreadsheet for use in Excel or Google Sheets.
Delete All Data
Permanently delete all works and shifts from this device. This cannot be undone.
